# Sistema de Imprenta en Java (POO)

Proyecto desarrollado en Java que simula el funcionamiento básico de una imprenta, aplicando principios de Programación Orientada a Objetos como interfaces, clases abstractas y polimorfismo para el manejo de distintos tipos de documentos.

## Objetivo del proyecto
Diseñar una solución orientada a objetos que permita modelar el proceso de impresión de documentos, aplicando abstracción y polimorfismo para facilitar la extensión del sistema sin modificar su lógica principal.

## Conceptos aplicados
- Programación Orientada a Objetos (POO)
- Interfaces
- Clases abstractas
- Polimorfismo
- Encapsulamiento
- Separación de responsabilidades

## Funcionalidades
- Modelado de distintos tipos de documentos (currículos, libros, informes)
- Implementación de comportamientos de impresión mediante interfaces
- Uso de polimorfismo para procesar diferentes documentos de forma genérica
- Ejecución del flujo de impresión desde una clase principal

## Tecnologías utilizadas
- Java
- Programación Orientada a Objetos

## Contexto
Este proyecto forma parte del proceso de aprendizaje de Java y está enfocado en fortalecer los fundamentos de diseño orientado a objetos necesarios para el desarrollo de aplicaciones backend más complejas.

## Posibles mejoras
- Incorporar persistencia de datos
- Aplicar este diseño en una arquitectura de capas
- Integrar el proyecto en un contexto de API REST utilizando Spring Boot
