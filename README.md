# Sistema de Imprenta en Java (POO)

Proyecto en Java que demuestra el uso de interfaces, clases abstractas y
polimorfismo mediante un sistema de impresión de documentos como currículos,
libros e informes.

## Conceptos aplicados
- Programación Orientada a Objetos
- Interfaces
- Clases abstractas
- Polimorfismo
- Encapsulamiento

Proyecto realizado como parte del aprendizaje en Java.
